# Handswork
Handswork是Java语言的架手架，封装常用的类和方法，后期将集成常用的web窗口。

## usage
```
    <repositories>
        <repository>
            <id>me.duanyong</id>
            <url>https://raw.github.com/duanyong/handswork/snapshot</url>
            <snapshots>
                <enabled>true</enabled>
                <updatePolicy>always</updatePolicy>
            </snapshots>
        </repository>
    </repositories>
```
